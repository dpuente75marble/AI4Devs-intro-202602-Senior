# Reverse String - Prompts Used

## Chatbot used

ChatGPT (GPT-4o)

---

## Prompt 1

Crea una página web simple en HTML y JavaScript que permita introducir una cadena de texto y que la invierta.

Requisitos:

- Debe tener un input de texto.
- Un botón para invertir la cadena.
- Mostrar el resultado en pantalla.
- Implementar la lógica en un archivo script.js separado.

---

## Prompt 2 (Nivel opcional)

Modifica la solución anterior para que:

- El botón solo aparezca cuando el texto tenga más de 3 caracteres.
- La cadena se invierta en tiempo real mientras el usuario escribe.
- Mantener también la funcionalidad del botón.
